Manual paste helpers for VBA editor

1. Create or open the standard module named ModuleLocalization and paste ModuleLocalization.for-vbe.bas.
2. Open the frmLetterCreator code window and replace the full code-behind with frmLetterCreator.code-for-vbe.bas.
3. Save in VBA editor, then export modules back through the modified VbaModuleManager.

These files are UTF-8 with BOM to preserve Cyrillic text in common Windows editors.
